# Angular8

### Project Technical Structure
***
*  Angular8
*  Angular/http
*  Rxjs


### Install

***
project address: (`git clone`)

```
git clone https://github.com/caolong2008023303/Pets.git
```
install node_modules with `npm`

```
npm install
```
***
### Run

start server (http://localhost:4200)

```
npm run start
```

publish production

```
npm run build
```
